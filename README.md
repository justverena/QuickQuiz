# QuickQuiz

## Описание проекта

**QuickQuiz** — это real-time платформа для создания и проведения онлайн-викторин.  
Проект ориентирован на образовательные сценарии (уроки, тесты, интерактивные квизы) и поддерживает одновременное участие множества пользователей в одной игровой сессии.

Ключевая особенность проекта — **использование WebSocket-соединений** для мгновенной передачи вопросов, ответов и обновлений результатов без перезагрузки страницы.

---

## Основные возможности

- Создание и управление квизами
- Запуск игровых сессий в реальном времени
- Подключение участников по коду сессии
- Мгновенная отправка ответов через WebSocket
- Live-обновление очков и таблицы лидеров
- Поддержка ролей (преподаватель / студент)
- Хранение и анализ игровых данных
- Логирование действий пользователей

---

## Архитектура проекта

Проект построен по **микросервисной архитектуре**, где каждый сервис отвечает за свою зону ответственности.

### Микросервисы

| Сервис | Назначение |
|------|-----------|
| **Quiz Service** | Управление квизами, вопросами, игровыми сессиями и логикой игры |
| **Auth Service** | Аутентификация, регистрация пользователей, управление ролями |
| **Log Service** | Сбор и хранение логов действий пользователей и системных событий |

---

## Real-time взаимодействие

Для реализации real-time функциональности используются **WebSockets**:

- Подключение участников к игровой сессии
- Отправка вопросов всем игрокам одновременно
- Приём ответов без HTTP-запросов
- Обновление состояния игры (статус, таймер, очки)
- Live Scoreboard

WebSocket-логика реализована с помощью **Django Channels** и **Redis (Channel Layer)**.

## Масштабируемость

Проект изначально спроектирован с учётом масштабирования:

- Горизонтальное масштабирование WebSocket-серверов

- Stateless WebSocket consumers
- Возможность вынесения сервисов в отдельные контейнеры
